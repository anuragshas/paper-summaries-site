We introduce Speculative Speculative Decoding (SSD), a novel framework that parallelizes the previously sequential speculation and verification phases of speculative decoding, thereby significantly accelerating autoregressive inference for large language models.
Core Problem
Autoregressive decoding is inherently bottlenecked by its sequential nature, and even speculative decoding, while faster, retains a sequential dependence between the speculation phase and the verification phase.
Why it matters:
  • Modern AI applications demand fast inference, but sequential decoding cannot fully leverage parallel computation hardware.
  • Reducing inference latency is crucial for improving user experience and enabling real-time applications of large language models.
Concrete Example:In standard speculative decoding (SD), the draft model must wait for the target model's verification pass to complete before it can begin speculating for the next round, leading to idle time.
Key Novelty
A novel framework, Speculative Speculative Decoding (SSD), that parallelizes the sequential speculation and verification steps inherent in existing speculative decoding methods for large language models.
  • The core idea of Speculative Speculative Decoding (SSD) is to parallelize the speculation and verification steps of speculative decoding. While the verifier is processing a batch of tokens, the draft model proactively predicts multiple likely verification outcomes and pre-computes corresponding speculative token sequences, storing them in a 'speculation cache'.
  • If the actual verification outcome matches one of the pre-speculated outcomes in the cache (a 'cache hit'), the pre-computed tokens are immediately returned, eliminating the latency overhead typically associated with drafting. If it's a 'cache miss', the system falls back to a synchronous speculative decoding mechanism.
  • The paper introduces Saguaro, an optimized SSD algorithm that addresses three key challenges: predicting verification outcomes efficiently (Saguaro cache construction), balancing cache hit rate and acceptance rate (Saguaro sampling), and handling cache misses effectively (Saguaro fallback).
  • SSD ensures losslessness by falling back to ordinary speculative decoding when pre-speculated outcomes are not matched, and it allows the draft model to run on separate hardware, enabling true asynchronous operation.
Evaluation Highlights
  • Saguaro (optimized SSD) achieved an average speedup of 4.7x over autoregressive decoding on Llama-3.1-Instruct 70B across four diverse datasets (Alpaca, GSM8k, UltraFeedback, HumanEval).
  • Saguaro demonstrated an average speedup of 1.88x (up to 2x) compared to optimized speculative decoding baselines.
  • The method successfully pushed the throughput-latency Pareto frontier, indicating improved compute efficiency, especially at lower batch sizes.
  • Empirical evaluation confirmed the benefits of the geometric fan-out strategy for cache hit rates and decoding speed, particularly at higher temperatures.
  • Saguaro's adaptive fallback strategy, which dynamically selects the backup speculator based on batch size, was shown to be optimal and significantly improved performance at larger batch sizes.
Breakthrough Assessment
7.5/10
The paper introduces Speculative Speculative Decoding (SSD), a novel framework that directly addresses the sequential bottleneck within existing speculative decoding by parallelizing speculation and verification. The Saguaro algorithm, an optimized instantiation of SSD, provides principled solutions for cache construction, sampling, and fallback strategies. It demonstrates substantial practical speedups (up to 2x over optimized speculative decoding and 5x over autoregressive generation), pushes the throughput-latency Pareto frontier, and is theoretically well-grounded. This represents a strong advancement in LLM inference efficiency.
